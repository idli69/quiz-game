# Quiz Game

A simple browser-based quiz app built with HTML, CSS, and JavaScript.

## Features
- Multiple questions
- Progress bar
- answer feedback
- Final result screen with message

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript


## Project Structure
- `index.html` –  structure
- `style.css` – styling
- `script.js` – Quiz logic
